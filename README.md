# v2Whitelist ![visitors](https://visitor-badge.laobi.icu/badge?page_id=kiktor12358.v2whitelist)

**v2Whitelist** — это упрощенный форк v2rayNG для Android, превращенный в "VPN в один клик".

## Особенности
- **Минимализм**: Никаких сложных настроек. Только одна большая кнопка подключения.
- **Умное подключение**: Приложение само выбирает самый быстрый сервер на основе реальной задержки (RealPing).
- **Автоматизация**: Встроенная подписка обновляется автоматически. Вам не нужно искать конфиги.
- **Отказоустойчивость**: Если сервер перестал работать, приложение автоматически переключится на другой в течение 5 секунд.

## Как это работает?
1. При запуске приложение проверяет и обновляет встроенную подписку.
2. При нажатии на кнопку **Start**, приложение тестирует доступные серверы.
3. Выбирается сервер с лучшим пингом и устанавливается защищенное соединение.

## Как собрать?
Проект настроен для автоматической сборки через GitHub Actions. При каждом пуше в репозиторий генерируется свежий APK в разделе **Actions**.
